A `Licences` resource represents a licence purchased by the user for a hosted app.

## Licenses representation {: #resource }

### JSON

```json
{
  "kind": "chromewebstore#license",
  "id": string,
  "appId": string,
  "userId": string,
  "result": string,
  "accessLevel": string,
  "maxAgeSecs": long
}
```
